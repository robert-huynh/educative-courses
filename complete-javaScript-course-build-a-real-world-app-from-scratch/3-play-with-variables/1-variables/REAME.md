# Variable


## Properties

- name
- value 
- type

## Declare a variable
- Use `let` keyword

## Assign value to a variable

- Use `=` sign 
- Different mathematical equality

## Declare a constant variable

- Use `const` keyword

## Increment operator

- `variable++`

## Variable scope

- `let` and `const` are block-scoped (just available in code block and every sub-block)
